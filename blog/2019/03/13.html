The Return

13 Mar 2019 - entro

I’m back.

I’ve made a return to the blogosphere. I am experimenting with Jekyll, a Ruby extension that allows for fun and easy static site generation. We’ll see how long I stick with it, but for now, it seems pretty cool.

As I’m sure you’ve noticed, my website has received a bit of a makeover. With Jekyll, I have a lot more control over my website than my previous boilerplate, copypasta mess. I’d like to hear any thoughts on how it compares to the old, if you remember that! I am still working on it, and that is now an easier and more enjoyable task, so I hope I’ll stick with it a bit more. Also, if you are an old visitor, I revived my old travel blog, check it out!

Thanks for checking in, stay tuned!
