I guess I made a website

April 15, 2016 McWenker

Welp, it looks like I've done it. I actually made a website that doesn't look like garbage. Props to the dudes at HTML5 Boilerplate for making the beginning easy, thanks to my job at TattooNOW for giving me a trial-by-fire in stylesheets.

I've still gotta figure out some kind of content management, cause right now I'm updating all of this content manually in Sublime Text, which leaves a lot to be desired when I'm trying to have all these pages working together. I'm looking into Jekyll to work with my GitHub Page (definitely worth checking out both of these if you're interested in building a website) but I've been unmotivated to navigate the workaround for it on Windows. One day.

Been working on some music lately, among other things. You can check all of that in my post about it, or on my SoundCloud. I'm looking to make some posts about everything else this or next week, just to actually populate this page and see what is and isn't working.

Anywho, thanks for checking this place out! Hope you'll come back to visit, because it's only going to get weirder.
